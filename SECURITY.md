# Security

## Reporting a vulnerability

Please report security issues **privately** so we can fix them before disclosure.

- **Preferred:** Use [GitHub private vulnerability reporting](https://github.com/ofershap/agents-control-tower/security/advisories/new) for this repository.
- **Alternative:** Open a [private security advisory](https://github.com/ofershap/agents-control-tower/security/advisories) or contact the maintainer via the email listed in the repo (if visible) or via GitHub.

Do not open a public issue for security vulnerabilities.

## Supported versions

Security updates are applied to the latest release. Upgrade to the newest version to receive fixes.

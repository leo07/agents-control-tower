# Contributing

Contributions are welcome. Here's how to get started.

## Setup

```bash
git clone https://github.com/ofershap/agents-control-tower.git
cd agents-control-tower
npm install
```

## Development

```bash
npm run dev          # run the TUI in dev mode
npm run typecheck    # check types
npm test             # run tests
npm run build        # build for distribution
```

## What to work on

- Bug fixes - check the issue tracker
- New components or improved UX
- Better error handling and edge cases
- Tests for components and hooks
- Phase 2: Local agent hooks integration

## Submitting a PR

1. Fork the repo and create a branch
2. Make your changes
3. Run `npm run typecheck && npm test && npm run build`
4. Open a PR with a clear description of what and why
